## How to Run Locally:


**Prerequisites:**
Before running the project, make sure you have [Node.js (ver 22.0.0 or above)](https://nodejs.org/en/download) installed.


##### 1. Set Up [React Native and Expo](https://expo.dev/go)
##### 2. Clone this repository in desired local directory.
##### 3. In terminal, install dependencies using command:
> ```bash
> npx expo install
##### 4. Start the Expo development server using command: 
> ```bash
> npx expo start
##### 5. Run the app
- Scan the QR code with the Expo Go app (iOS/Android), or
- Press a to open Android emulator / i for iOS simulator.
